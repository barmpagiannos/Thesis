# Thesis
Μια ολοκληρωμένη μελέτη πάνω στην εφαρμογή της Βαθιάς Μάθησης στο Φυσικό Επίπεδο των Τηλεπικοινωνιών. Το project περιλαμβάνει την ανάπτυξη Autoencoders για End-to-End επικοινωνία, MIMO συστήματα, την ενσωμάτωση γνώσης πεδίου με Radio Transformer Networks (RTNs) και την αυτόματη ταξινόμηση διαμορφώσεων με χρήση CNNs πάνω σε ωμά IQ δείγματα.
