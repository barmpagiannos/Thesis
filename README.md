# Μια εισαγωγή στη Βαθιά Μάθηση για το Φυσικό Επίπεδο

Η παρούσα διπλωματική εργασία διερευνά τη μετάβαση από τα παραδοσιακά τηλεπικοινωνιακά συστήματα σε data-driven αρχιτεκτονικές με χρήση Deep Learning. Στον κώδικα υλοποιούνται Autoencoders που μαθαίνουν αυτόματα βέλτιστες διαμορφώσεις και καταπολεμούν παρεμβολές πολλών χρηστών χωρίς τη χρήση χρονομερισμού (TDMA). Παράλληλα, εξετάζεται η αντιμετώπιση πολυδιαδρομικών διαλείψεων Rayleigh μέσω Radio Transformer Networks (RTNs) και αναπτύσσονται Συνελικτικά Νευρωνικά Δίκτυα (CNNs) που υπερτερούν των παραδοσιακών αλγορίθμων στην ταξινόμηση σημάτων.

## Βασικά Χαρακτηριστικά
* **End-to-End Επικοινωνία:** Ανάπτυξη συστημάτων με χρήση Autoencoders για την αυτόματη ανακάλυψη βέλτιστων αστερισμών στο επίπεδο I/Q.
* **Συστήματα MIMO:** Επίλυση προβλημάτων παρεμβολής σε κανάλια πολλών χρηστών (Interference Channels), με θεαματική βελτίωση της φασματικής απόδοσης έναντι του κλασικού χρονομερισμού (TDMA).
* **Radio Transformer Networks (RTNs):** Ενσωμάτωση ειδικής γνώσης πεδίου (Informed ML) μέσω παραμετρικών μετασχηματιστών, για την ταχύτερη σύγκλιση και διόρθωση διαλείψεων σε πολυδιαδρομικά κανάλια.
* **Ταξινόμηση Διαμόρφωσης (Modulation Classification):** Εκπαίδευση CNNs σε ωμά IQ δείγματα για την αυτόματη ταξινόμηση 11 αναλογικών και ψηφιακών διαμορφώσεων.

## Ο Κώδικας
Όλα τα πειράματα, οι αρχιτεκτονικές των νευρωνικών δικτύων και οι οπτικοποιήσεις των αποτελεσμάτων (καμπύλες BLER, t-SNE διαγράμματα αστερισμών, Confusion Matrices) είναι διαθέσιμα στα Jupyter Notebooks (`.ipynb`) αυτού του αποθετηρίου. Λόγω της φύσης του Jupyter Notebook, μπορείτε να δείτε τα γραφήματα και τα αποτελέσματα χωρίς να χρειαστεί να τρέξετε τον κώδικα τοπικά.

## Δεδομένα (Dataset)
Τα δεδομένα που χρησιμοποιήθηκαν για την εκπαίδευση του Συνελικτικού Νευρωνικού Δικτύου (CNN) στο Κεφάλαιο της Ταξινόμησης Διαμορφώσεων είναι ανοιχτά και προέρχονται από το **RadioML dataset**. 

Μπορείτε να βρείτε και να κατεβάσετε το αυθεντικό dataset από την επίσημη ιστοσελίδα της DeepSig:
**[https://www.deepsig.ai/datasets/](https://www.deepsig.ai/datasets/)**
